Основное описание в AsciiDoc.
* Swagger: /EJournal/swagger/htmlSwag
* JavaDoc: /EJournal/javaDoc
* AsciiDoc: /EJournal/asciiDoc

CORS конфиг находится в ru.configs.WebMVCConfig.

Тестовый режим включается в ru.Main.test
Тестовый режим подключает тестового пользователя.
UUID-токен: 9693b2a1-77bb-4426-8045-9f9b4395d454, администратор с ником "nm12"

Для смены пользователя на тестовом токене перейти к запросу "/auth/infCon/"
и в теле запроса указать логин нужного пользователя.
Пример тела:
{
	"login": "testLogin"
}

Обычно у всех разделов есть эндпоинт с которого обязательно начинать работу, а иногда и несколько.
Перед описанием эндпоинта пишется [start]
